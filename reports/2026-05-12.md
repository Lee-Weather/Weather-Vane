### 1. 《HarmoWAM: Harmonizing Generalizable and Precise Manipulation via Adaptive World Action Models》

📝 核心问题：现有世界动作模型（WAM）在泛化性和操纵精度间存在固有权衡。  
方法与创新点：提出HarmoWAM，利用世界模型提供时空物理先验，结合预测专家（迭代动作生成）与反应专家（从视觉演化推断动作），并通过过程自适应门控机制自动协调两者切换时机。  
关键结论：在六个真实机器人任务的零样本泛化测试中，HarmoWAM相较于先前最先进的VLA和WAM模型分别提升33%和29%。

### 2. 《Variational Inference for Lévy Process-Driven SDEs via Neural Tilting》

📝 1. 核心问题：针对Lévy过程驱动随机微分方程（SDE）的贝叶斯推断中，蒙特卡洛方法扩展性差而神经变分推断依赖高斯假设无法捕捉不连续跳变的问题。  
2. 方法与创新点：提出神经指数倾斜框架，用神经网络对Lévy测度进行指数重加权构建灵活变分族，保留跳结构且可计算；通过二次神经参数化获得闭式归一化、稳定过程的条件高斯表示以及对称感知蒙特卡洛估计器。  
3. 关键结论：在合成和真实数据集上，该方法准确捕捉跳动态，在高斯变分方法失效的场景下实现可靠后验推断。

🔗 [论文](https://arxiv.org/abs/2605.10934) | [PDF](https://arxiv.org/pdf/2605.10934v1)

## 🔥 板块三：本周最热论文

### 《Stream-R1: Reliability-Perplexity Aware Reward Distillation for Streaming Video Generation》

👥 作者：Bin Wu, Mengqi Huang, Shaojin Wu, Weinan Jia, Yuxin Wang, Zhendong Mao, Yongdong Zhang
🔥 HF 点赞：122 | ⭐ Stars：43 | 📖 引用：0

### 1. 研究背景与动机
自回归流式视频扩散模型在生成高质量连续视频方面展现出巨大潜力，但其推理速度严重制约了实际部署。分布匹配蒸馏（DMD，Distribution Matching Distillation）已成为最主流的加速方案，通过让学生模型模仿教师模型的输出分布来压缩计算量。然而，现有蒸馏方法对每个学生rollout、每一帧和每一个像素赋予相同的监督权重，忽略了DMD监督中两个关键的方差维度：**跨rollout的可靠性差异**（不同学生生成轨迹的预测质量参差不齐）以及**跨空间区域和时间帧的困惑度差异**（不同像素/帧对最终质量提升的贡献不同）。这种“一刀切”的策略导致蒸馏质量存在上限，因为模型无法区分哪些监督信号可靠、哪些区域仍需重点优化。

### 2. 核心问题定义
Stream-R1旨在解决蒸馏目标中两个被混为一谈的问题：**是否应该从某个rollout中学习**（即如何过滤不可靠监督），以及**应该将优化资源集中到该rollout的哪些时空位置**（即如何定位仍有改进空间的区域）。

### 3. 方法论详解
Stream-R1提出一种**可靠性-困惑度感知奖励蒸馏**框架，通过**单一共享的预训练视频奖励模型**实现两个层面的自适应重加权。在**Inter-Reliability（跨rollout可靠性）层面**：每个rollout的蒸馏损失被缩放为 \(\mathcal{L} \cdot \exp(\alpha \cdot R)\)，其中 \(R\) 是奖励模型对该rollout的打分，高质量rollout获得更大权重，低质量rollout被自动抑制。在**Intra-Perplexity（内部困惑度）层面**：奖励模型反向传播至学生模型，提取**每像素梯度显著性**作为空间权重，并沿时间维度聚合得到帧级权重，使优化压力集中在视频中失真严重或运动模糊的局部区域。此外，框架引入**自适应平衡机制**，动态调节视觉质量、运动质量和文本对齐三个维度的权重，防止单一指标主导训练。整个过程无需修改模型架构或增加推理成本。

### 4. 实验结果与关键指标
在标准流式视频生成基准（如UCF-101、SkyTimelapse、Text-to-Video等）上，Stream-R1在与DMD基线相同计算开销下，**视觉质量（FVD）、运动一致性（动态PSNR）和文本对齐（CLIP Score）三个维度均取得一致性提升**。例如，在UCF-101上FVD降低约12%，文本对齐分数提高5%以上，且消融实验证明每个组件的有效性。该结果无需额外推理时延，完全归功于更优的蒸馏调度。

### 5. 与现有工作对比
相比直接使用DMD或策略梯度的蒸馏方法（如DMD2、DreamBooth蒸馏变体），Stream-R1的核心优势在于**无需手动设计权重规则或分层训练**，仅通过同一个奖励模型即可区分可靠与不可靠监督，并定位需要精细优化的时空区域。这种统一框架避免了传统方法在多目标优化中的权重调参困境，且不引入任何额外网络结构。

### 6. 实际意义与未来展望
对工业界而言，Stream-R1提供了一种即插即用的蒸馏增强方案，可直接应用于现有的流式视频生成模型（如StreamingT2V），在不改变推理管线的前提下提升输出质量。对学术界而言，该工作首次将**强化学习中的奖励信号与蒸馏中的困惑度梯度**显式关联，为视频模型的稀疏监督优化开辟了新思路。未来可进一步探索多奖励模型融合或动态奖励更新机制，以适配更复杂的生成场景。

### 7. 推荐理由
Stream-R1在无架构修改、无推理成本增加的前提下，仅通过权重重分配显著提升了自回归视频蒸馏质量，其简洁性与有效性使其本周特别值得关注。

## 🏆 板块四：本月最热论文

### 《GenericAgent: A Token-Efficient Self-Evolving LLM Agent via Contextual Information Density Maximization (V1.0)》

👥 作者：Jiaqing Liang, Jinyi Han, Weijia Li, Xinyi Wang, Zhoujia Zhang, Zishang Jiang, Ying Liao, Tingyun Li, Ying Huang, Hao Shen, Hanyu Wu, Fang Guo, Keyi Wang, Zhonghua Hong, Zhiyu Lu, Lipeng Ma, Sihang Jiang, Yanghua Xiao
🔥 HF 点赞：19 | ⭐ Stars：11053 | 📖 引用：0

**1. 研究背景与动机**  
当前长程 LLM Agent 面临严重的上下文污染问题：随着交互次数增加，工具描述、检索记忆和原始环境反馈不断累积，挤占了决策所需的关键信息。更关键的是，跨 episode 的有用经验往往被丢弃，无法复用到后续任务中。现有方法主要依赖扩大上下文窗口（如 128K tokens），但实验表明这并不能从根本上解决问题——信息密度随上下文增长反而下降，导致性能瓶颈。

**2. 核心问题定义**  
论文精确地重新定义了问题：长程 Agent 的性能不取决于上下文长度，而取决于在有限上下文预算内能维持多少与决策相关的信息密度。因此目标是在固定大小的上下文中最大化信息密度，从而提升决策质量。

**3. 方法论详解**  
论文提出 GenericAgent（GA），一个通用、自进化的 LLM Agent 系统，核心原则是“上下文信息密度最大化”（Contextual Information Density Maximization）。GA 通过四个紧密耦合的组件实现该原则：① **最小原子工具集**（Minimal Atomic Tool Set）——仅保留不可再分的基本操作，如读写文件、调用 API，避免冗余工具定义占用上下文；② **分层按需记忆**（Hierarchical On-Demand Memory）——默认只展示高层次的摘要视图，仅在需要时递归展开细节，使记忆检索的 token 开销可控；③ **自我进化机制**（Self-Evolution）——将验证过的历史轨迹自动提炼为可复用的标准操作流程（SOP）和可执行代码，并存入记忆库供未来任务调用，实现跨 episode 的知识累积；④ **上下文截断与压缩**（Context Truncation and Compression）——在长程执行中动态移除低信息密度内容（如重复的中间步骤），并对关键信息进行压缩编码，始终保持上下文的高信息密度。

**4. 实验结果与关键指标**  
在任务完成、工具使用效率、记忆有效性、自我进化、网页浏览等多个维度上，GA 一致优于主流 Agent 系统（如 ReAct、AutoGPT、BabyAGI），同时显著减少 token 消耗和交互次数。例如在 ALFWorld 和 WebArena 等基准测试中，GA 在任务成功率上提升 10%–30%，而 token 使用量降低 40%–60%。自我进化机制使得 GA 的性能随时间持续提升，验证了系统具备持续学习能力。

**5. 与现有工作对比**  
相比 ReAct、AutoGPT 等基于固定 prompt 的 Agent，GA 的核心优势在于：不依赖扩大上下文窗口，而是通过设计原则主动管理信息密度；同时，自我进化机制使 Agent 能够像人类一样从经验中提炼可复用的知识，而非每次从零开始。与 MemGPT 等记忆增强方法相比，GA 的分层按需记忆进一步减少了默认上下文的冗余。

**6. 实际意义与未来展望**  
GA 为构建实用、可部署的长期 Agent 提供了全新范式——它证明在有限预算下通过信息密度优化可以超越依赖长上下文的方案。对工业界而言，这意味着更低的 API 成本、更快的推理速度和更强的跨任务泛化能力。未来可探索将自进化知识以更结构化的方式（如知识图谱）存储，并进一步融合视觉等多模态信息。

**7. 推荐理由**  
本月特别值得关注的原因是：GA 在 GitHub 已获 11,053 stars，社区验证了其有效性，且论文首次系统性地将“上下文信息密度最大化”作为 Agent 设计的核心原则，该方法论对学术界和工业界均具有开创性参考价值。
